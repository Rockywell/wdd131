My Solar System

Site Name

I chose my-solar-system.com because it represents and encourages others to learn more about the solar system we all live in.

Site Purpose

The site focuses on teaching about the wonders of the solar system. With a small focus on encouraging audiences to expand their education in astronomy, to dig deeper and learn more.

Scenarios

  • Why is this important to know?
  • What's an interesting fact about the solar system?
  • Where/how can I learn more?

Color Schema

  • Primary: Black will be used for background colors and heading elements (depending on the contrast ratio.)
  • Secondary: White will be used for heading elements and paragraph text.
  • Accent: Rebeccapurple will be used for small elements like borders and navigation elements and hover effects.

Typography

I will be using Montserrat for my heading font and Arial for my body font. Sans-Serif will be the default fallback font.
